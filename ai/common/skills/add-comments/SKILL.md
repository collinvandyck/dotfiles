# Overview

Your task is to add comments to the specified code. If no code is specified, try to get it from the attached
IDE context.

# Guidance

- The comments should only be added if valuable. For example, do not add a comment if it's obvious what the code is doing.
- Comments should follow the comment style found elsewhere.
- If the code is complex, the comment should explain at a very high level what the code does and potentially why it's needed.
