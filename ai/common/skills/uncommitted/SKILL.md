# Overview

Your task is to review the uncommitted changes (staged and unstaged) for correctness.

## Guidance

- Do not make changes to the code
- Only comment on what the diff shows. Do not make claims about unchanged code (e.g., "X doesn't have this yet") without reading the file to verify — the current state may already include it from a prior commit.
